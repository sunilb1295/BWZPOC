# SAP Build Work Zone Test Applications

This workspace contains two SAP Fiori applications for SAP Build Work Zone:

- **postloginplugin** - A shell plugin that enhances the Fiori Launchpad with custom header actions and API integrations
- **demoappbwz** - A demo application showcasing multi-language support and basic Fiori functionality

## Quick Start

Navigate to the respective folder and run:
```bash
npm install
npm start
```
